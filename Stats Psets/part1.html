Topic 1: Population Parameters and Sample Estimates

We are often interested in estimating "true" (population) parameters using sample data. The population parameters are features of the "true" model or data generating process. For example, the population average value of the roll of a fair die is (1 + 2 + 3 + 4 + 5 + 6)/6 = 3.5.

Recall the samples from the introductory example: {4, 1, 6, 2, 5, 4, 1, 3, 6, 2} and {3, 5, 2, 6, 6, 3, 4, 2, 3, 5}. What are the sample averages in these samples? (4 + 1 + 6 + 2 + 5 + 4 + 1 + 3 + 6 + 2)/10 = 3.4 and (3 + 5 + 2 + 6 + 6 + 3 + 4 + 2 + 3 + 5)/10 = 3.9. Notice that neither sample average is equal to the population average of 3.5.

Despite this, we use sample averages like the ones above to estimate population averages when population averages are unknown. The same is true for other population parameters.
