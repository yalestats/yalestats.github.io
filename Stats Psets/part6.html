Question 2: Hypothesis Testing

The last (and perhaps most important) component of hypothesis testing is the actual test itself. This can take one of two forms, with the first being the most common:

1. Comparing the p-value \(p\) to the significance level. We commonly use \(\alpha\) to represent the significance level.

a. If \(p < \alpha\), you reject the null hypothesis.

b. If \(p > \alpha\), you fail to reject the null hypothesis because of insufficient evidence to support the alternative hypothesis.

2. Comparing the test statistic \(t\) to a critical value \(t_{crit}\).

a. If \(|t| > |t_{crit}|\), you reject the null hypothesis.

b. If \(|t| < |t_{crit}|\), you fail to reject the null hypothesis because of insufficient evidence to support the alternative hypothesis.

Before going on to an application, let's define the significance level and critical value.

The significance level of a hypothesis test is a value chosen by you, the analyst, before performing any data analysis. The most common are \(\alpha = 0.01, 0.05, \text{ and } 0.1\). The significance level is the maximum probability you allow for committing a Type I error, which is the probability of rejecting the null hypothesis when it is actually true. If you choose a significance level of \(\alpha = 0.05\), it means you are willing to accept a 5% chance of rejecting the null hypothesis when it is true (a 5% chance of a Type I error).

The critical value is the value of the test statistic used in hypothesis testing to determine whether you should reject or fail to reject the null hypothesis. It depends on the chosen significance level \(\alpha\). A smaller \(\alpha\) will result in a larger critical value.
