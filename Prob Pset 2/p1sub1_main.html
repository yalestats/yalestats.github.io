Sub Problem: Expected Value

Let’s work through an example to see exactly how the expected value formula works.

Suppose you have a bag with three balls and each ball is labeled with a number. Let’s say we have balls labeled “0”, “4”, and “5”. The expect values asks us “If we were to draw a ball at random, what is the average value of the number written on the ball that we would draw?”. We may be interested in such a value if, for example, someone had agreed to pay us dollars equal to the number on the ball that we draw. The expected value could then be interpretted as "on average, the dollar value of this offer".
