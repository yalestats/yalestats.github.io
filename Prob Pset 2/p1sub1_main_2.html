Sub Problem: Expected Value cont.

How did we compute the expected value just now? If we followed the definition from class, we probably multiplied each outcome value by the probability of occurring which is equal to 1/3 and performed the calculation 1/3*0 + 1/3*4 + 1/3*5.

Another way to arrive at the correct answer that you may have been led to by your intuition was to average the numbers on the three balls and compute (0+4+5)/3. This gives the correct answer, and it’s easy to see that it is equivalent to the calculation performed above. If you followed your intuition, this is probably what you did.

In fact, an average is exactly the same thing as an expected value, and we can build on this intuition to compute expected values in more complex environments.
