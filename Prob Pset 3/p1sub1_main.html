This sub-problem will work through some examples to build this intuition.

Did you notice that the expected value of Y was exactly two greater than the expected value of X and that the variance (and thus standard deviation) was the same? This is not a coincidence! We could have added 100 or subtracted 15 (i..e added -15) and the same result would occur: the expected value would “shift” according to the number that we added and the variance would be unchanged.

If we think a little bit about the geometry/visual of what is going on, this result become intuitive. When we added a number, say 4, to a random variable, we are “shifting” all the possible outcome 4 “spaces” to the right. This increases the expected value. But because we’ve shifted all outcomes by the same amount, we haven’t changed the spread of the random variable at all, so the variance remains unchanged.
